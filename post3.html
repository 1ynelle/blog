Post 3 - MDN HTML Research

November 10, 2020

Today I'd like to share a few interesting HTML tidbits I've learned from the MDN website.

When it comes to responsive images, adding the srcset and sizes attribute to the image element is useful for smaller screens when large files are not needed. Srcset defines the set of images to choose from while "sizes" specify which image to download depending on the screen size.

Similar to images, other multimedia content such as YouTube videos and Google Maps can be embedded through the iframe element. You can try it out by going on YouTube, clicking share â†’ embed to copy the code and paste it in your HTML.

In addition to ordered lists, and unordered lists, there's the description list which is useful for marking up words and definitions or questions and answers. Other useful text markup elements include the abbreviation, address, and inline quote elements.

Hopefully these tips gave you a deeper understanding of HTML. You can learn more by visiting MDN.
