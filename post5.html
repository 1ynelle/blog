Post 5 - Web Design Research 1

November 19, 2020

Edoardo Cavazza's article on "Modern CSS Techniques To Improve Legibility" offers some useful tips to make text on the web more accessible. I read this article to learn ways I can improve my blog design which is quite text-heavy. Below are a few of the tips I found interesting.

Different fonts, though set at the same size, can visually vary in size (depends on the x-height). By matching the heights of the fonts' lowercase letters using the font-size-adjust property, the browser can render the fonts at the same size visually. This provides a smoother transition into a fallback font and helps when spacing with rems and ems.

Other suggestions include adjusting the tracking to enhance readability and using calc, CSS variables, and a color scheme media query to change colors depending on the contrast with the background. Increasing the spacing between characters and for large text sizes makes a greater impact on readability than a legible font.

I find it interesting how small details can enhance a design and make reading easier. I like how the author gives practical examples of where these CSS properties are used and how designers and developers can improve their websites. The article showed me the importance of prioritizing accessibility and user preferences as much as form and functionality.
